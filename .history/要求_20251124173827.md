# 一、项目技术栈
- 后端使用 Flask + SQLAlchemy
- 数据库可选择 SQLite（默认）或 MySQL
- 采用分层结构：routes（API 层）、services（业务层）、models（数据库）、utils（工具）

# 三、功能模块说明
1. 用户反馈提交 API
   - 路由：POST /api/feedback/add
   - 字段：content、contact
   - 服务：FeedbackService.add_feedback()
   - 自动捕获 IP

2. 后台登录 API
   - 路由：POST /admin/login
   - 使用 JWT 或自定义 token
   - 配置中的固定账号可用于登录

3. 后台反馈分页查询
   - 路由：GET /admin/feedback/list?page=1&size=10&keyword=xxx
